# KSBA 홈페이지 개편 내역

## 2.0.0 · 2026-07-21

- PC 1200px 기준과 모바일 360px 기준을 함께 고려한 반응형 레이아웃 재구축
- 모든 제목·본문에 `clamp()`와 단계별 모바일 글자 크기 적용
- 협회 소개, 주요 사업, 회원, 표준, 교육, 소식, 문의를 독립 페이지로 분리
- 설립 준비 단계와 확정 전 사업을 명확히 구분하도록 문장 전면 수정
- 법인·인증·자격 실적을 과장할 수 있는 표현 제거
- 네이비·틸·골드 기반 협회형 디자인 시스템 적용
- 모바일 메뉴, 키보드 접근성, 본문 바로가기, 포커스 상태 적용
- 서버 저장 없는 이메일 문의 기능 구현
- sitemap, robots, manifest, 보안 헤더, 404 페이지 추가
- 외부 패키지를 제거해 Cloudflare 의존성 설치 지연 문제 최소화
